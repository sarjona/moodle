Yii PHP Framework 1.1.22
-------------
https://github.com/yiisoft/yii

Instructions to import 'Yii PHP Framework' into Moodle:

1. Download the latest release from https://github.com/yiisoft/yii/releases

2. Copy 'framework/web/helpers/CJSON.php' into 'lib/yii/framework/web/helpers/CJSON.php'.
For now, this is the only file used, so the other files are ignored and not included.

3. Update version number in admin/lib/thirdpartylibs.xml
